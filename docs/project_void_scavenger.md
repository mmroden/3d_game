Building a Descent-style 6DOF roguelike space shooter for Mac with SBS stereoscopic 3D rendering.

Stack: Godot 4 + godot-rust (game logic in Rust, Godot for rendering/editor/export)
Assets: Quaternius Modular Sci-Fi MegaKit (CC0) + Sci-Fi Essentials Kit + Kenney Space Kit
Target platforms: macOS, Linux (Steam Deck)
Controller support: Xbox/PS/Steam Deck via GCController/Godot input

Features planned: procedural level generation from modular corridor pieces, silly name generator for items/stations, lootbox ship upgrades, Hades-style meta-progression (later).

**Why:** No good SBS 3D games available on Mac. Building one.

**How to apply:** v0.1 scope is: one proc-gen level, 6DOF flight, one enemy, one lootbox, SBS toggle, controller input.
